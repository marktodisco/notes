# App Requirements

1. Read any number of markdown files from a directory.
2. Parse YAML metadata in markdown header.
     1. start datetime
     2. end datetime
     3. tags
3. Combine markdown files in chronological order.
4. Display notes in a Dash web app
5. Display table of contents on the left-hand side
6. Group notes by years, quarters, months, and weeks.
7. Search notes for keywords and tags.
